# Judge notes

Private Scratch Wallet is intentionally small and focused.

It is not a generic privacy demo. It applies confidentiality to a concrete DeFi agent problem: hiding autonomous risk policies while keeping public auditability.

The repo is no-server and GitHub Pages-ready. The Sepolia contract provides a public registry for policy commitments and decisions. The `nox/` folder documents the confidential policy flow and provides a replacement point for production Nox runtime calls.
