# Submission

## Project name

Private Scratch Wallet

## One-liner

A no-server GitHub Pages dApp that keeps autonomous DeFi risk policies private with a Nox-compatible confidential flow while publishing auditable decisions on Ethereum Sepolia.

## Description

Private Scratch Wallet extends the Scratch Wallet concept for the iExec WTF Hackathon. It addresses a key privacy issue in autonomous DeFi: risk policies such as bankroll, max trade size, max daily loss and execution thresholds should not be public. The app builds a confidential policy commitment, runs a Nox-compatible decision flow, and publishes only commitments and final decisions to Sepolia.
